# 关于我

## 我是谁

一个在 IT 行业摸爬滚打了十多年的"老油条"。从物理服务器到虚拟化，从 IDC 到云原生，从脚本小子到自动化运维，这十多年见证了太多技术变迁。

**什么都会一点，什么都不精通** — 这就是我最真实的写照。Linux 系统调优懂一些，网络设备配置能上手，数据库也折腾过几种，中间件也部署过好几轮，甚至前端、Python、K8s 都沾过边。但要说哪一样是专家级别的，还真不敢说。

## 这个站点是干什么的

这个网站最初只是我的个人笔记，用来记录工作中踩过的坑和解决问题的思路。后来发现，很多技术细节用过就忘，每次都要重新查资料，太浪费时间了。

所以这个站点的定位很简单：

1. **自己的"第二大脑"** — 记不住的就查这里，整理过的知识随时翻
2. **技术速查手册** — 210+ 篇速查文档，覆盖从 AI 到快捷键的 15 个分类
3. **踩坑记录** — 那些年踩过的坑、填过的洞，都记下来免得下次再踩
4. **知识沉淀** — 把散落在各处的笔记整理成体系，方便自己，也方便有同样需求的人
