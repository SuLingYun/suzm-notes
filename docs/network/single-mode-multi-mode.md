# 单模光模块与多模光模块的区别

## 1. 什么是单模与多模光模块

**单模（SM）光模块**使用局限性小，适用于远距离数据传输。
**多模（MM）光模块**可以传输多种模式的光，适用于近距离传输。

## 2. 区别对比

### （1）波长不同
- 多模光模块：工作波长一般为 **850nm**
- 单模光模块：工作波长一般为 **1310nm、1550nm**

### （2）传输距离不同
- 单模光模块：常用于远距离传输，可达 **150~200km**
- 多模光模块：多用于短距离传输，**2km 以下**

### （3）光纤类型不同
- 多模光纤（MMF）：纤径一般为 **50/125μm** 或 **62.5/125μm**
- 单模光纤（SMF）：纤径为 **9/125μm**

### （4）光源不同
- 单模光模块：光源是 LD 或光谱线较窄的 LED
- 多模光模块：光源是发光二极管或激光器

### （5）应用范围不同
- 单模光模块：多用于传输速率较高、距离较远的线路，如城域网建设
- 多模光模块：多用于短距离传输，网络节点和接头较多的传输场景

### （6）价格不同
- 单模光纤比多模光纤便宜
- 但单模光模块价格远高于多模光模块

## 3. 注意事项

1. 使用单模光模块时，需确保实际接收光功率小于过载光功率，必要时配合光衰减器使用
2. 使用单模光模块时，接收功率要有余量
3. 多模光模块只能发送一种波长的信号，不能与复用器配合使用
4. 多模光模块只限于在多模光纤上运行
5. 单模设备可在单模和多模光纤上运行，但在多模光纤上不能保障效果
