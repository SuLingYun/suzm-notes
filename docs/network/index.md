# 网络运维笔记

本目录汇集了网络运维相关的技术笔记，涵盖防火墙配置、WAF 防护实践、网络故障排查等内容。

## 涵盖内容

- **路由交换**：华为交换机运维、H3C IRF 堆叠配置
- **防火墙**：防火墙策略配置、规则管理
- **WAF 防护**：Web 应用防护、安全策略
- **网络配置**：Linux 网络配置、Bonding、路由设置
- **内网穿透**：FRP 配置、隧道搭建
- **故障排查**：网络连通性测试、性能诊断

## 常用网络命令速查

| 命令 | 用途 | 常用参数 |
|------|------|---------|
| `ping` | ICMP 连通性测试 | `-c <次数>` `-i <间隔>` |
| `traceroute` | 路由路径追踪 | `-n`（不解析域名）`-T`（TCP 模式） |
| `ss` | 查看 socket 统计 | `-tuln`（监听端口）`-ant`（连接状态） |
| `ip addr` | IP 地址配置 | `show` / `add` / `del` |
| `ip route` | 路由表管理 | `show` / `add` / `del` |
| `tcpdump` | 流量抓包 | `-i eth0 -nn port 80` |
| `nmap` | 端口扫描 | `-sT -p 1-1000 <目标>` |
| `dig` | DNS 查询 | `@8.8.8.8 example.com A` |
