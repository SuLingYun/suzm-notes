# 自动化运维

自动化运维是云时代的核心竞争力。本文档整理了配置管理、CI/CD、监控告警等领域的实战经验。

## 文章列表

- [Ansible 入门与实战](/automation/ansible) — 从零开始配置管理，批量部署、配置同步、滚动更新
- [Jenkins CI/CD 流水线搭建](/automation/jenkins) — Pipeline 编写、代码审查、自动部署、通知集成
- [Prometheus 监控体系搭建](/automation/prometheus) — Exporter 配置、Alertmanager 告警规则、Grafana 仪表盘
- [Zabbix 企业级监控实战](/automation/zabbix) — 自动发现、触发器配置、邮件/钉钉告警、性能调优
- [Shell 脚本自动化实战](/automation/shell-scripts) — 日志分析、服务巡检、备份脚本、定时任务

## 工具速查

| 工具 | 用途 | 典型场景 |
|------|------|----------|
| **Ansible** | 配置管理 | 批量部署、配置同步、应用发布 |
| **Jenkins** | CI/CD | 代码构建、测试、部署流水线 |
| **Prometheus** | 监控告警 | 容器监控、服务发现、多维告警 |
| **Zabbix** | 监控告警 | 传统服务器、网络设备监控 |
| **GitLab CI** | CI/CD | Git 仓库集成、自动化测试 |
| **Cron** | 定时任务 | 数据备份、日志清理、周期性巡检 |
